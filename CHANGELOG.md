# 更新日志

## [1.2.0] - 2026-06-23

### 新增功能

- **对话历史持久化**：对话记录自动保存到 localStorage，刷新页面不丢失
- **AI 回复一键复制**：鼠标悬停（桌面端）或直接可见（移动端）显示复制按钮，点击即可复制纯文本话术
- **清空对话**：桌面端和移动端工具栏均添加清空按钮，一键重置训练
- **品牌开场动画**：加载时 Flora AI 品牌优雅渐入，带绿色线条生长动画
- **浮动花瓣背景**：8 片半透明花瓣在页面中缓缓飘落，营造花艺氛围
- **底部标签图标**：移动端底部导航添加 SVG 图标（对话气泡/时钟/文档）

### 视觉升级

- **玻璃拟态增强**：卡片增加内发光效果，更有层次深度
- **卡片悬停交互**：知识卡片悬停微抬升，话术卡片悬停右滑
- **品牌渐变标题**：Flora AI 标题从深色渐变到鼠尾草绿
- **发送按钮升级**：鼠尾草绿渐变 + 阴影浮起效果
- **打字指示器**：从简单圆点升级为玻璃气泡样式
- **场景导航动效**：悬停/激活时平移和渐变指示条
- **场景色彩条**：移动端场景选择器色彩条增加渐变效果
- **全局微交互**：所有可点击元素增加微位移和阴影反馈

### 移动端优化

- **场景滑动条**：添加右侧渐变遮罩，提示可滚动
- **花语速查**：全新卡片设计，大色块圆点 + 花名 + 花语 + 箭头引导
- **话术模板**：场景标签 + 上下文标签 + 预览文字三层结构
- **内容区滚动**：修复为内部滚动，头部/底部标签固定不动
- **安全区域适配**：底部标签适配 iPhone 安全区域
- **输入区域**：更大圆角，发送按钮纯渐变无边框
- **复制按钮**：触屏设备默认半透明可见

### 性能优化

- **API 请求体优化**：移除每次请求携带全部场景数据（~12KB），请求体缩减约 90%
- **场景查找**：`findScene` 从嵌套循环 O(n×m×k) 优化为 Map 查找 O(1)
- **DOM 缓存**：添加 `$()` 和 `isDesktop()` 工具函数，减少重复查询
- **格式化函数**：`formatAIText` 3 个 heading replace 合并为 1 个正则
- **代码精简**：消除冗余代码，文件行数减少 26%（1566 → 1157 行）

### AI 体验改进

- **系统提示词优化**：增加回复原则、互动模式和边界说明
- **话术风格**：引导 AI 生成更自然口语化、可直接用于门店的销售话术

### Bug 修复

- 修复开场动画最后一帧左移问题
- 修复开场动画绿色线条变细问题
- 修复开场动画结束后页面布局偏移
- 修复移动端花语速查和话术模板底部大片空白
- 修复移动端页面整体滚动问题，改为内容区内部滚动
